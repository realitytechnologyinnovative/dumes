# D-DUMES SHELTER LIMITED Website

Official website for D-DUMES SHELTER LIMITED - Transforming Dreams into Reality.

## About

This is the official website for D-DUMES SHELTER LIMITED, a leading Nigerian housing development company dedicated to addressing the critical need for affordable, low-cost housing across the nation. Established in 2011 and registered with RC: 943183.

## Technology

This website is built with [Jekyll](https://jekyllrb.com/) and designed to be hosted on [GitHub Pages](https://pages.github.com/).

## Local Development

To run this site locally:

1. Install Ruby and Bundler
2. Install dependencies:
   ```bash
   bundle install
   ```
3. Run Jekyll:
   ```bash
   bundle exec jekyll serve
   ```
4. Open [http://localhost:4000](http://localhost:4000) in your browser

## Project Structure

- `_config.yml` - Jekyll configuration
- `_data/` - Data files (company info, services, projects, etc.)
- `_includes/` - Reusable components (header, footer, etc.)
- `_layouts/` - Page layouts
- `index.html` - Homepage
- `about.html` - About page
- `services.html` - Services page
- `projects.html` - Projects page
- `contact.html` - Contact page
- `css/`, `js/`, `images/` - Static assets

## Deployment

This site is configured for GitHub Pages. Simply push to the `main` branch (or `gh-pages` branch) and GitHub Pages will automatically build and deploy the site.
